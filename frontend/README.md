# Football Predictions UI (NextJS)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ennyolar1996/betwey_sure.git
   cd betwey_sure
   cd frontend
   ```

## Install dependencies:

```bash
$ npm install
```

## Run the app:

```bash
  $ npm run dev
```

## Open it on your browser:

```bash
  $ curl "http://localhost:3000"
```
